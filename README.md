# Bedolaga VPN Bot — Deploy config

Конфиги для деплоя на VPS (shadowpro.tech).

## Содержимое

- `Caddyfile` — обратный прокси (бот + кабинет + webhooks)
- `.env.bot.example` — переменные бота (скопировать в bot/.env)
- `.env.cabinet.example` — переменные кабинета (скопировать в cabinet/.env)

## Репозитории

- Bot: https://github.com/Fr1ngg/remnawave-bedolaga-telegram-bot
- Cabinet: https://github.com/BEDOLAGA-DEV/bedolaga-cabinet
